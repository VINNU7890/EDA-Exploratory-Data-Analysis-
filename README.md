 # SQL Exploratory Data Analysis (EDA) Project

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned layoffs dataset using MySQL. After completing the data cleaning process, the dataset was analyzed to identify trends, patterns, and insights related to layoffs across different companies and years.

The goal of this project was to use SQL queries to analyze company-wise and year-wise layoffs, rank companies based on total layoffs, and generate meaningful business insights from the data.

---

## Tools & Technologies Used
- MySQL
- SQL

---

## Analysis Performed
- Total layoffs analysis using SUM()
- Company-wise layoff analysis
- Year-wise layoff analysis using YEAR(Date)
- Ranking companies based on total layoffs
- Identifying top companies with the highest layoffs each year
- Trend analysis using SQL queries and CTEs

---

## SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM)
- Common Table Expressions (CTEs)
- Ranking Functions
- Date Functions
- Filtering & Sorting

---

## Key Insights
- Google recorded one of the highest layoffs in 2023
- Meta and Amazon showed major layoffs in 2022
- Uber had the highest layoffs in 2020 in the dataset
- Layoff trends varied significantly across different years and companies

---

## Project Outcome
The analysis helped transform cleaned data into meaningful insights by identifying layoff trends, ranking companies based on layoffs, and understanding year-wise patterns in the dataset.
